### 5. source order、role order、stable source paths、family-local exact path dedupe、16 record limit、160 excerpt limit

**结论: PASS**

- Role order: `investor_return` → `holder_structure` → `share_change` → `subscription_redemption` → `income_distribution`，与 plan §4 一致。
- Source order within each role: `sections` → `paragraph_blocks` → `table_blocks` → `table_blocks[*].cells`，由 `_select_investor_experience_candidate_evidence` 中调用顺序确定。
- Stable source paths: `sections[i]`、`paragraph_blocks[i]`、`table_blocks[i]`、`table_blocks[i].cells[j]`（j 为原始 tuple index）。
- Cell scan order: `sorted(enumerate(table.cells), key=lambda item: (item[1].row_index, item[1].column_index))`，`source_field_path` 保留原始 tuple index `cell_index`。
- Family-local exact path dedupe: `seen_paths: set[str]` 在 `_select_investor_experience_candidate_evidence` 中管理。
- 16 record limit: `_INVESTOR_EXPERIENCE_CANDIDATE_LIMIT = 16`，主循环中有 `if len(records) >= limit: break`，最终截断 `records[:limit]`。
- 160 excerpt limit: 复用 `_CANDIDATE_EXCERPT_LIMIT = 160`，通过 `_truncate()` 实现。
- `test_investor_experience_selector_orders_dedupes_limits_and_truncates` 验证了 16 条限量、role order、source order、path 稳定性、dedupe 和 excerpt ≤ 160。

- Token false positives remain possible because locator matching does not prove semantic field correctness. Owner: later field-extraction/source-truth gate; S6-E mitigates by keeping public status `missing`.
- `income_distribution` role labels may be suppressed by same-path first-record-wins when an earlier role matches the same source path. Owner: future field-extraction/source-truth design; accepted as non-blocking for S6-E per controller judgment A2.
- `subscription_redemption` cell guard context excludes `cell_text` / `cell_text_normalized` per binding amendment A1. This means if a cell's text is `申购赎回` (strong token) it still matches directly, but if it is only `申购` (generic), it needs parent table heading/caption/row-label/column-header context containing a guard token. This is the intended behavior per the plan and controller judgment.
- Candidate excerpts can contain numeric-looking text (percentages, shares, dividends, 申购/赎回 quantities, holder counts). Owner: S6-E implementation keeps excerpts out of `value` and `anchors`; tests assert no value leak.
- Existing S6-A/B/C/D regression tests remain unchanged and passing. No refactor was performed as part of S6-E.
